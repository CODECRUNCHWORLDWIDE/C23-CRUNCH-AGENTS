# Week 22 — Challenges

The exercises drill the pieces — the diagram, the retrieval interface, the memory tiers. **The challenge is the sprint itself.** You ingest the real 10 GB corpus, land hybrid retrieval and the three memory tiers behind their interfaces, measure them on the gold set and the regression test, and ship the Mermaid diagram and the 6-page architecture document — the actual Sprint A deliverable the capstone is graded on.

## Index

1. **[Challenge 1 — The Sprint A foundation](challenge-01-sprint-a-foundation.md)** — ingest the corpus, land hybrid retrieval (measured Recall@5/MRR on the 100-question gold set) and the three memory tiers (passing the turn-38 regression test), draft the agent contracts, and ship the architecture diagram + 6-page document. (~150+ min; this is the sprint)

This challenge is not optional — it *is* Sprint A, the foundation the rest of the capstone (weeks 23–24) is built on. The Phase II milestone (hybrid retrieval over a real corpus, chunking A/B'd, three memory tiers, a 6-page architecture memo) is graded on exactly this. The skill — integrating a course's worth of components into a coherent, interfaced, measured foundation, and documenting it well enough to defend at a sealed review — is the capstone skill, and Sprint A is where you prove you have it. Get the foundation right and Sprint B is assembly; get it wrong and Sprint B is a re-architecture under deadline.
